 STEP 2 — ADD YOUR OPENAI KEY IN VERCEL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

THIS IS THE ONLY KEY YOU MUST SET. Everything else is optional.

  1. Go to vercel.com → your project
  2. Click "Settings" tab
  3. Click "Environment Variables" in the left menu
  4. Click "Add New"
  5. Name:  OPENAI_API_KEY
     Value: sk-proj-xxxxxxxxxxxxxxx  (your full OpenAI key)
  6. Make sure all three checkboxes are checked:
       ✓ Production
       ✓ Preview  
       ✓ Development
  7. Click "Save"
  8. Go to "Deployments" tab → click the three dots on latest → "Redeploy"
     (environment variables only take effect after a redeploy)

 HOW THE NEW ARCHITECTURE WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before (broken):  Browser → Vercel → Railway → OpenAI
After (working):  Browser → Vercel → OpenAI

Vercel Serverless Functions (/api/*.js files):
  /api/health        → GET  → returns server status
  /api/parse         → POST → reads uploaded PDFs, returns profile JSON
  /api/opportunities → POST → calls OpenAI + job APIs, returns opportunities
  /api/message       → POST → calls OpenAI, returns outreach message

These run automatically. No server to manage. No monthly hosting cost.
Vercel's free tier gives you 100,000 function invocations per month.
